# 15. 三数之和 (3Sum)

## 题目描述

给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。

请你**返回所有和为 0 且不重复的三元组**。

**注意：** 答案中不可以包含重复的三元组。

**示例 1：**
```
输入: nums = [-1,0,1,2,-1,-4]
输出: [[-1,-1,2],[-1,0,1]]
解释: -1 + -1 + 2 = 0，-1 + 0 + 1 = 0
```

**示例 2：**
```
输入: nums = [0,1,1]
输出: []
```

**示例 3：**
```
输入: nums = [0,0,0]
输出: [[0,0,0]]
```

## 约束条件

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

## 解题思路

### 方法一：排序 + 双指针

**核心思想：**
- 先对数组进行排序
- 固定一个数 `nums[i]`，然后使用双指针在 `[i+1, n-1]` 范围内寻找两个数，使它们的和等于 `-nums[i]`
- 通过跳过重复元素来避免重复的三元组

**关键步骤：**
1. 对数组 `nums` 进行升序排序
2. 遍历 `i` 从 `0` 到 `n-3`：
   - 如果 `nums[i] > 0`，由于数组已排序，后面的数都大于0，直接 break
   - 如果 `i > 0` 且 `nums[i] == nums[i-1]`，跳过以避免重复
   - 使用双指针 `left = i+1`，`right = n-1`
   - 当 `left < right` 时：
     - 计算 `sum = nums[i] + nums[left] + nums[right]`
     - 根据 `sum` 调整指针位置
     - 当 `sum == 0` 时，将 `[nums[i], nums[left], nums[right]]` 加入结果，并跳过重复元素

## 代码实现

### 方法：排序 + 双指针
Java:
```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> list = new ArrayList<>();
        Arrays.sort(nums);
        for(int i=0;i<nums.length-2;i++){
            //排除第一个数大于0的情况
            if(nums[i]> 0) break;
            // 如果这个数与上一个数一样，可能会出现找到两次一样的情况
            if(i>0 && nums[i-1]==nums[i]) continue;
            int left = i + 1;
            int right = nums.length-1;
            // 双指针找另外两个数
            while(left < right){
                int sum = nums[i]+nums[left]+nums[right];
                if(sum == 0){
                    list.add(Arrays.asList(nums[i],nums[left],nums[right]));
                    //匹配成功之后再去匹配看看有没有一样的
                    while(left < right && nums[left] == nums[left+1]) left++;
                    while(left < right && nums[right] == nums[right-1]) right--;
                    left++;
                    right--;
                }
                if(sum > 0) {right--;}
                if(sum < 0) {left++;}
            }
        }
        return list;
    }
}
```
Python:
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        length = len(nums)
        res = []
        nums.sort()
        for i in range(length - 2):
            # 如果nums[i] > 0 则可以直接舍弃
            if nums[i] > 0:
                break
            # 如果与上一个数重复，则直接舍弃
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            # 双指针找另外两个值(可能有多个)
            l = i + 1
            r = length - 1
            while l < r:
                sum = nums[i] + nums[l] + nums[r]
                if sum == 0:
                    res.append([nums[i], nums[l], nums[r]])
                    while l < r and nums[l] == nums[l + 1]:
                        l = l + 1
                    while l < r and nums[r] == nums[r - 1]:
                        r = r - 1
                    l = l + 1
                    r = r - 1
                elif sum > 0:
                    r = r - 1
                elif sum < 0:
                    l = l + 1
        return res
```



## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 排序 + 双指针 | O(n²) | O(1) 或 O(n)（取决于排序算法） |

## 关键点总结

- ✅ 先排序是避免重复的关键
- ✅ 固定一个数，双指针找另外两个数
- ✅ 跳过重复元素避免重复三元组
- ✅ 优化：当 `nums[i] > 0` 时可直接退出
- ⚠️ 注意边界条件和指针移动的时机
